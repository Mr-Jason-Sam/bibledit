Text Area

This is the area where the text of the Bible can be edited

Bible Notes Area

This is the area for the Bible notes, that is, the footnotes, endnotes and crossreferences that are part of the Bible.

Project Notes Area

This is the area for the project notes (called translation notes in Paratext).

Tools Area

This is the area that contains various tools.

It provides tools and shows one of these areas:

- References Area and the Quick Reference Area

- Styles Area

- Note Tools Area

Changing between the areas

Of course, changing between the areas using the mouse is easy. Just click in the area.

The keyboard can be used too to change from one area to another.
