Bibledit

Bibledit needs some libraries and tools which are not installed on most standard installations of Linux. Therefore you need to install these before installing Bibledit. In most cases it is sufficient to install a C++ compiler, the development packages of libgtk2, libsqlite3, libxml2, git-core, libxul, and all their dependencies. Then install Bibledit.

Bibledit also can be installed on the Macintosh. It is the same story as with Linux: A couple of libraries are needed to get it going.

As for Windows, somebody is working on a Windows port.

Windows Outpost

Bibledit needs the Outpost to interact with Windows related stuff. If there is no such stuff, there is no need to install the Outpost.
