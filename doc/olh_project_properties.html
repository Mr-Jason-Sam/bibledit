Project Properties

In this dialog you can set the properties of a project.

The dialog is accessible through menu File / Project / Properties.

You can change the name of the project.

By default a project is editable, but you can set it non-editable. In that case the project cannot be changed or deleted.

You can add or remove books.

When adding books, you can either import a book from file, or add a book using an empty template.

You can set the versification system of the project, and the language. Both of these get their data from http://scripturechecks.sourceforge.net.
