Syntax Notes

Required spaces are indicated with _. Parameters are indicated in ALLCAPS or by a special character such as the hash mark #. Optional or suggested information is shown as normal text in (parentheses).

Markers are indicated in the following manner

  • E.g. "\id " means that this marker has no endmarker.
  • E.g. "\f *" signals that this marker has an obligatory endmarker.
  • E.g. "\ft (*)" signals that this marker has an optional endmarker.
