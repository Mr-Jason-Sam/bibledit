Introduction

It is assumed that a standard installation of the operating system is used.

Insert the installation media.

Click Applications, at the top.

Click System.

Click Synaptic Package Manager.

Provide the user's password, if asked for it.

Bibledit can be installed from a package or from source. Make your choice below.

From a package

In the Synaptic Package Manager, click Settings, then Repositories.

Click Add.

Click Custom.

Enter this line:

deb http://packages.sil.org/ubuntu edgy main

Click Add Channel.

Select the Authentication tab.

Download http://packages.sil.org/ubuntu/dists/silarchive_2006.asc and save it somewhere.

Click Import Key File.

Select the silarchive_2006.asc you downloaded and click OK.

Click Close to get back to Synaptec.

Press the Reload button and let the update finish.

Search for bibledit, mark it for installation, and click button Apply.

When it asks to apply the changes, click Apply.

Close the window when through.

Open a terminal by clicking Applications, Accessories, Terminal.

Proceed with running Bibledit.

From source

In the Package Manager, search for "g++". At the right side of the window mark the package for installation. When it asks Mark additional required changes, click Mark.

Do the same for "libgtk2.0-dev", for "libsqlite3-dev", and for "subversion".

Apply the changes.

The installer will download some packages from the installation media or the internet and takes a while to update the system.

Close the window when through.

Open a terminal by clicking Applications, System, Terminal.

Install Bibledit.

Run Bibledit

Run Bibledit.

Install XEP.
