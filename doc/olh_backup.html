This assistant is accessible through menu File / Backup.

A variety of data can be backed up through this assistant.

Bible

When backing up up a Bible, normally the Bible now focused in the editor will be backed up. You can choose another one.

Notes

When backing up Notes, all the available notes will be backed up.

Resource

When backing up a Resource, normally the Resource now focused will be backed up. You can choose another one.

Everything

This backs up really everything Bibledit knows about.

Depending on the amount of data that you have , it may take a good while to create this backup.

This type of backup is recommended for safekeeping all your data, and when you'd like to transfer all your data from one computer to the other.

If Bibledit starts, and notices that it has no data, it will ask for a complete backup. A complete backup is one that was made with the "Everything" option.
