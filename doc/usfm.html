USFM

Unified Standard Format Markers (USFM) are markers that are put in the text to indicate the book, where a chapter starts, the verse, titles and so on. A marker generally starts with a backslash (\) at the beginning of a line of text in the editor.

The markers are divided into several groups.

  • Identification
  • Introduction
  • Titles
  • Headings
  • Chapters and Verses
  • Paragraph Elements
  • Poetry Elements
  • Table Elements
  • Footnote and Endnote Elements
  • Cross References
  • Extended Study Notes
  • Special Text and Character Styles
  • Peripheral Materials

Environmental markers.

The above information has been compiled from version 2.05 of the USFM 2 standard.

Bibledit supports several of the markers.

By editing the stylesheet of a project, the number of supported markers can be extended. Completely new markers can be added too, but this is not recommended. It is better to stick to the USFM standard. The stylesheet can be edited under Preferences > Stylesheets.
