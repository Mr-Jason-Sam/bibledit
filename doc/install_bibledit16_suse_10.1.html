(Open)Suse 10.1

These instructions assume Bibledit will be installed on a standard installation of the operating system, with KDE desktop.

Insert the installation media.

Click the Suse icon, at the bottom left.

Click System, then Configuration, then Install Software.

Search for "gtk2-devel", and tick it.

Click button Install.

Add a privileged user, and give the password of root, and close the dialog.

Click button Install again.

The installer will take a while to update the system. Close the window when through.

In the same way search for and install "sqlite-devel" and "gcc-c++" also.

Open a terminal by clicking the Konsole icon, the third one from the left at the bottom.

Install Bibledit.

Run Bibledit.

Install XEP.
