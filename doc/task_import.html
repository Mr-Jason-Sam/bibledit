Importing text

Bibledit can import existing USFM files.

In the menu choose File / Project / Import / USFM or other formatted text.

Select the directory that contains the existing USFM files.

In most cases it is enough to then click OK.

The books will be imported.

Also raw text can be imported. See File / Project / Import / Raw text.

See also: Import, Import text dialog.

Standard texts

Some standard texts have been provided by the community. These are listed below.

UBS Greek Text. Comment: This server is in Jakarta, and extremely slow if you are not on that side of the globe. For use in Bibledit, I would recommend using the wonderful SIL Gentium font for the text display font.

Word English Bible. Comment: World English Bible in USFM form.

King James Bible. Comment: KJV in USFM form. Link at bottom of the left frame.
