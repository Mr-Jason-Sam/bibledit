Install Bibledit Windows Outpost

Download package bwoutpostxx.exe to the home directory (xx is to be replaced with the actual version number).

In the terminal, to verify the package is there, type command

ls

The program should be installed through Wine. Ensure Wine has been installed properly, see Install Wine.

In the terminal type

wine bwoutpostxx.exe

Follow the installation instructions on the screen and complete the installation.

Bibledit will recognize it and on startup it will start the Outpost too.

Note: If you install the Outpost using CrossOver Office, then you'll need to adjust the settings for the Outpost through menu Preferences / Windows Outpost.

If the Windows stuff runs on a Windows machine, then the Oupost can be installed there too.
