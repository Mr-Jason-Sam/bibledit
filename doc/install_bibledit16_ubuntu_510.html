Ubuntu 5.10

It is assumed that a standard installation of the operating system is used.

Development tools and libraries have to be installed too.

Insert the installation CD or DVD.

Click System, at the top left.

Click Administration

Click Synaptic Package Manager.

Provide the user's password, if asked for it.

In the Package Manager, click Development.

At the right side, mark the following package for installation: g++.

When it asks Mark additional required changes, click Mark.

At the left side, scroll down a bit and click Libraries - Development.

At the right side, mark the following package for installation: libgtk2.0-dev.

When it asks Mark additional required changes, click Mark.

Scroll down a bit and mark libsqlite3-dev for installation.

When it asks Mark additional required changes, click Mark.

Click button Search, and search for "make".

In the search results, scroll down, and mark the following package for installation: make

In the same way, install package "subversion".

Click button Apply.

When it ask to apply the following changes, click Apply.

The installer will download some packages from the CD, DVD or internet and take a while to update the system. Close the window when through.

Open a terminal by clicking Applications, System Tools, Terminal.

Install Sqlite3.

Install Bibledit.

Run Bibledit.

Install XEP.
