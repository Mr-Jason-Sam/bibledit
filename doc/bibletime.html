BibleTime

If BibleTime is installed, Bibledit will detect that, and the two will work together in harmony.

You can set Bibledit to send references to Bibletime. This means that when the translator is working in Bibledit on e.g. Mark 1:10, then BibleTime will show Mark 1:10 too. When the translator then goes to the next verse, BibleTime will do the same.

In Bibledit, go to menu Preferences - Reference exchange, and set Bibledit to send references to BibleTime.

You can also set Bibledit to receive references from BibleTime. If the translator works in BibleTime, and goes to the next verse, Bibledit will do the same.

This feature needs BibleTime 1.5.2 or newer.

Bibledit can also search in BibleTime modules.

BibleTime uses the SWORD engine to display its resources. Bibledit can export a project to a SWORD module, see Project.

For information on how to install BibleTime see the BibleTime site.
