Fedora Core 6

These instructions assume the operating system has been installed as a default system.

The standard installation needs some extra tools, such as a compiler and libraries.

Insert the installation DVD in the drive.

Click "Applications", then then "Add/Remove Software".

Give the password for root.

In the Package Manager, click Search.

Search for, and tick the following packages: gcc-c++, gtk2-devel, sqlite-devel.

Click Apply.

Click Continue.

Updating the system takes a while to complete.

After completion, click button OK.

In the Package Manager, click button Quit.

To open a terminal, click "Applications", then "Accessories" and then "Terminal".

Install Bibledit.

Run Bibledit.

Install XEP.
