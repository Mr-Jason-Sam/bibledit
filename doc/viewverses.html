Viewing verses

Bibledit can display one verse of one or more projects, all in one window.

This is in short what "viewing verses" is all about.

The user can view verses by ticking the Verses entry in the View menu.

A new window displays, where the verses are displayed.

The window shows a "Settings" expander at the top, and white space below that.

Clicking on "Settings" a few buttons appear, and a list above that. That list might be empty, and does not show in that case.

Click the "Add" button to add a project.

Once a project has been added to the window, the verse that is now in the editor will also show in the "Verses" window. For example, if the user is editing Matthew 2.5, and has added project "Matthew Poole", then Matthew 2.5 from the commentary of Matthew Pools will display. Of course, this only works if the project "Matthew Poole" is available.

Click the "Add" button again to add another project. And so on.

A project can be added more than once.

Select a project, or more than one, and then click the "Delete" button to delete that project from the "Verses" window.
