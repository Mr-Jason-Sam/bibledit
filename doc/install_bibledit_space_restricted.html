Install bibledit from source on a space restricted device

Insert a USB stick or SD card into the device so as to increase the available storage space. If the device does not automatically mount, mount it manually. In the example below the device is available at /media/usbdisk.

Download package bibledit-x.x.tar.gz (x.x is to be replaced with the actual version number).

Move it to the /media/usbdisk directory

To change to that directory, type

cd /media/usbdisk

In the terminal, to verify the package is there, type

ls

To unpack bibledit type

tar zxf bibledit-x.x.tar.gz

To enter the newly created directory, type

cd bibledit-x.x

To configure bibledit, type

./configure

Watch many messages, but no errors.

To compile bibledit, type

make

Watch many messages, but no errors.

To install it, type

sudo make install

Watch many messages, but no errors.

To go back to the home directory, type

cd
