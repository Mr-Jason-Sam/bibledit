Planning

Bibledit can produce a planning. This shows the various tasks that have been planned for the project, when they start and finish, and how far they have progressed.

There are four places in the Bibledit menu that are relevant to planning.

1. Preferences / Reporting. Here is where the various tasks are set and the times they take to complete one chapter.

2. Preferences / Planning. Here is where to enter when the project started.

3. Edit / Planning. This dialog shows a list of tasks that have been planned for the project. The user can add and delete tasks, or reorder them.

4. View / Planning. This produces the report. It contains an estimated date of completion, and a list of planned tasks with their duration, when they start and finish. The duration of a task is calculated based on the number of days per chapter this task takes, multiplied by the number of chapters in the book this task is about.
