Debian 3.1

The installation below works for Bibledit 2.2, but not for later versions. Bibledit 2.3 and up don't run on Debian 3.1.

These instructions assume the operating system has been installed as a default system with a graphical desktop.

This installation needs a compiler and libraries.

Insert the installation media.

Run Applications - System Tools - Synaptic Package Manager.

In the package manager, search for libgtk2.0-dev, mark it for installation, and mark the additional required changes too.

Do the same for package g++.

Click Apply and let the package manager finish.

To open a terminal, click the Terminal icon at the top.

Install Sqlite3.

Install Bibledit.

Run Bibledit.

Install XEP
