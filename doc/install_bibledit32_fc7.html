Fedora Core 7

These instructions assume a standard installation of the operating system.

Insert the installation DVD in the drive.

Click "Applications", then then "Add/Remove Software".

Give the password for root.

In the Package Manager, click Search.

Search for, and tick the following packages: gcc-c++, gtk2-devel, sqlite-devel, libxml2-devel, git-core, firefox-devel, enchant-devel.

Click Apply.

Click Continue.

Updating the system takes a while to complete.

After completion, close the Package Manager.

Set SELinux to Permissive, through "System", "Administration", "Firewall and SELinux".

To open a terminal, click "Applications", then "System Tools" and then "Terminal".

Install bibledit.

Run Bibledit.

Install the optional components.
