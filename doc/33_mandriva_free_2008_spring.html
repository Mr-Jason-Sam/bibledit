Mandriva Free 2008 Spring

These instructions assume Bibledit will be installed on a standard installation of the operating system with KDE desktop.

Insert the installation media.

Click the Menu, at the bottom left.

Click Install and Remove Software.

Provide the password, and continue till you are in Software Management.

Search for "libgtk2+2.0_0-devel", and tick it, and accept any dependencies it wishes to install as well.

Click button Apply.

The installer will take a while to update the system. Close the window when through.

In the same way search for and install "make", "gcc-c++", "git-core", "gtkhtml2-devel", "libcurl-devel".

Open a terminal by clicking the Suse icon, then Terminal Program.

Install Bibledit.

Run Bibledit.

Install the optional components.
