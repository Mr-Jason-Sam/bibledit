Fedora Core 4

These instructions assume that the operating system was installed as a Personal Desktop, with all defaults chosen.

This setup needs additional tools, such as a compiler and libraries.

Insert the installation DVD in the drive.

Click "Desktop", then "System Settings", then "Add/Remove Applications".

Give the password for root.

In Package Management, tick Development Tools and GNOME Software Development.

Click Update.

Click Continue.

Updating the system takes a while.

After completion, click button OK.

In Package Management, click button Quit.

To open a terminal, click "Applications", then "System Tools" and then "Terminal".

Install Sqlite3.

Install Bibledit.

Run Bibledit.

Install XEP
