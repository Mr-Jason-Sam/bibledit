Debian 4.0 (Etch)

The instructions assume that Debian was installed with all defaults, including the graphical Desktop.

Run Desktop - Administration - Synaptic Package Manager.

Click Search and search for "bibledit". Mark it for installation and click Apply to install it.

Run Bibledit and optionally install XEP.

This would give the version of Bibledit that ships with Debian.

If a newer version is desired, do this:

In the Synaptic Package Manager, search for "g++". At the right side of the window mark the package for installation. When it asks Mark additional required changes, click Mark.

Do the same for the following packages:

- libgtk2.0-dev

- libsqlite3-dev

- subversion

- libpopt-dev

- libxml2-dev

- libgtkhtml3.8-dev

Apply the changes.

The installer will download some packages from the installation media or the internet and takes a while to update the system.

Close the window when through.

Open a terminal by clicking Applications, Accessories, Terminal.

Install bibledit.

Run bibledit.

Install XEP.

Open a terminal.
