Reporting

Bibledit can produce a status report. This report shows the status of that project, that is, how much of the project has been completed.

There are three places in the Bibledit menu that are relevant to reporting.

Preferences / Reporting, for setting the tasks assigned to the project.

Edit / Status, for setting which tasks have been completed.

View / Status, for producing the report.

See each of the above three items for more information.

Once the report has been produced, if a task has been completed it gets a tick (✔). If a task has not been done yet, it gets a cross (✗). If both of these marks are given (✗ ✔), it means that the task has been partially done.

The CSV export allows the user to make graphs that look better than the ones that come with Bibledit.
