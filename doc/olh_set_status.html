Set Status

This dialog is accessible from menu Edit / Status, then clicking any of the buttons there.

It shows all the tasks that a project consists of.

Put a tick before any of these to indicate that this task has been done.

If there is no tick, then this status has yet to be done.

At times there is another symbol that indicates that the task has not been consistently completed. Such a state occurs when for example the status of a chapter is displayed, and not all the verses of that chapter have a particular task completed. Some have, and others don't.
