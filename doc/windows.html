Bibledit on Windows

Bibledit runs on Windows too. All information about it is available on the Bibledit wiki, see section Build Instructions for MinGW. or section Install Bibledit on Windows. Very recent automatically created builds are available from the Buildbot at Bibles International. This autobuilder builds every 6 hours if something has changed in the interim. This

means the Windows binaries are now available for the head of the master branch in the source repository.
