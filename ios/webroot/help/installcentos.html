gettext("Install Bibledit 1.9 on CentOS 6.4")

gettext("Follow the steps below to install Bibledit.") gettext("It needs some basic knowledge of working with Linux servers.")

  1. gettext("Open a terminal.")
  2. gettext("Install Apache, PHP5 and MySQL.")
  3. Set up the firewall to enable incoming web traffic.

  4. gettext("In the terminal issue the following commands:")

    $ yum install git unzip rcs php-mysql php-mbstring php-xml php-posix php-sqlite

    $ service httpd restart

  5. gettext("Select the default settings for anything to be set up.")

  6. gettext("Download Bibledit:")

    $ wget http://download.savannah.nongnu.org/releases/Bibledit/source/Bibledit-1.9.zip

  7. gettext("Install Bibledit in folder /var/www/html as user root:")

    $ cd /var/www/html

    $ unzip ~/Bibledit-1.9.zip

  8. gettext("Make everything readable and writable to the web server user through the following commands")

    $ chown -R apache:apache Bibledit

  9. gettext("Bibledit will be accessible through:") http://website.org/Bibledit.
  10. gettext("Open the web address in the browser, and follow the steps on the screen to configure Bibledit and log in.")
  11. gettext("Enable the use of .htaccess files in Apache to secure some Bibledit folders.") gettext("Set") AllowOverride All gettext("in file") httpd.conf.
