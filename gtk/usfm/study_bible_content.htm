Project Configuration

Content for a study Bible consists of 2 or 3 related Paratext projects, depending on the scope and specifications of the intended publication.

 

1.One project contains the reference text, which is the scripture text for which the notes are being created.
 

This project is configured to use the standard Paratext usfm.sty stylesheet.
It it typically configured with editing disabled.
 

2.A second notes project contains the following content types:
 
·Extended book introductions.
·Division and section introductions.
·Study notes.
·Reflective questions.

 

This project is configured to use the usfm_sb-notes.sty stylesheet.

 

3.A optional third sidebars project can contain larger sections of topical content which are associated with a general area in the scripture reference text, but usually not a specific verse or word.
 
This project is configured to use the usfm_sb-sidebars.sty stylesheet.
