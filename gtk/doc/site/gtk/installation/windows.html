Windows

Windows Installation

Bibledit-Lite is a version forked off Bibledit-Gtk. Recent automatically created builds are available from the Buildbot at Bibles International. This autobuilder builds every 6 hours if something has changed in the interim. This means the Windows binaries are now available for the head of the master branch in the source repository.

Just in case the server at Bibles International is down, a copy of the 22 March 2011 build is available at http://download.savannah.gnu.org/releases/bibledit/binaries/. It is bibledit-win32(-opt).zip.
