Bibledit 4.6 on Ubuntu 12.04 LTS

It is assumed that a standard installation of the operating system is used.

Connect to the installation media.

Open a terminal, and do the following:

$ sudo apt-get install git
$ git clone git://git.savannah.nongnu.org/bibledit
$ sudo apt-get install build-essential libgtk2.0-dev
$ sudo apt-get install rcs
$ sudo apt-get install sqlite3 libsqlite3-dev
$ sudo apt-get install libxml2-dev
$ sudo apt-get install libenchant-dev 
$ sudo apt-get install libgtkhtml3.14-dev
$ sudo apt-get install libgtksourceview2.0-dev
$ sudo apt-get install libwebkit-dev
$ cd bibledit/gtk
$ ./configure
$ make
$ sudo make install
