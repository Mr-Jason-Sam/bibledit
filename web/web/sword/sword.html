Sword

Folder "basic-1" contains a Sword module generated from OSIS generated from basic USFM through the Python usfm2osis.py script.

Folder "basic-2" is the same as above, but using the Perl usfm2osis.pl script.

Folder "full-1" contains a Sword module generated from OSIS generated from full USFM through the Python usfm2osis.py script.

Folder "full-2" is the same as above, but using the Perl usfm2osis.pl script.

Each folder contains a zipped Sword module. This Sword module can be downloaded, and then be installed in the Sword Bible program.

The Sword utilities are maintained by the CrossWire Bible Society.

If the administrator has set up Sword module repositories, then the following four remote sources will be available:

Type Host Directory
1 FTP _host_ /_bible_/sword/basic-1
2 FTP _host_ /_bible_/sword/basic-2
3 FTP _host_ /_bible_/sword/full-1
4 FTP _host_ /_bible_/sword/full-2

The Python usfm2osis.py script is not yet "production ready". Whatever use you make of the OSIS file generated by usfm2osis.py is at your own risk.
